### Cannot Log In as Admin
**Problem:** "Invalid credentials" error
**Solution:**
- Use email: `fk.envcal@gmail.com`
- Use password: `Admin123!`
- Make sure you're on the admin login page: `/login/admin`

### Cannot Log In as Vendor
**Problem:** "Account pending approval" message
**Solution:**
- Use the active vendor account: `test.vendor@purple-soul.com` / `VendorTest123!`
- OR log in as admin first and approve the pending vendor

### Forgot Which Account to Use
**Solution:**
- For admin access: `fk.envcal@gmail.com` / `Admin123!`
- For vendor access: `test.vendor@purple-soul.com` / `VendorTest123!`
- Both are pre-approved and ready to use

### "Unauthorized" or "Access Denied"
**Problem:** Trying to access wrong dashboard type
**Solution:**
- Admin accounts must use `/login/admin`
- Vendor accounts must use `/login/vendor`
- They are separate authentication flows
